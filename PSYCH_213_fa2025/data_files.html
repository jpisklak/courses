  • Do not try to open them in Excel or other spreadsheet programs (they will crash your computer).
  • Load them directly into R.
  • Use the tidyverse read_csv() function — it is much faster than base R’s read.csv().
  • If using Colab, uploading through the sidebar may take time since each file is about ~80 MB.
  • A faster option is to load the data directly from GitHub using the raw file URL, e.g.:
    read_csv("https://raw.githubusercontent.com/...")
