How to Create a CSV File

Without Spreadsheet Software | Using Spreadsheet Software

Creating a CSV File Without Spreadsheet Software

Suppose this is the data you want to be a CSV file:
Student Coffee Cups Per Day Grade
Alice 1 65
Bob 2 72
Charlie 3 80
Dave 4 88
Eve 5 95

1. Open a plain text editor

  • Windows: Use Notepad or Notepad++
  • macOS: Use TextEdit (switch to plain text mode: Format → Make Plain Text)
  • Linux: Use Gedit, Nano, or any text editor

2. Enter data in CSV format:

  • Each line represents a row in a spreadsheet
  • Commas are used to delimit (i.e., sepearate), cells.
Student,Coffee_Cups_Per_Day,Grade
Alice,1,65
Bob,2,72
Charlie,3,80
Dave,4,88
Eve,5,95

3. Save the file with a ".csv" extension

  1. Go to File → Save As
  2. Choose All Files as the format (if applicable)
  3. Name the file with a .csv extension (e.g., data.csv)
  4. Choose UTF-8 encoding (if available) to avoid issues with special characters.


Creating a CSV File Using Spreadsheet Software

Method 1: Microsoft Excel

  1. Open Excel and start a new blank worksheet.
  2. Enter your data in a tabular format, with column headers in the first row.
    3. Student Coffee Cups Per Day Grade
    Alice 1 65
    Bob 2 72
    Charlie 3 80
    Dave 4 88
    Eve 5 95
  3. Click File → Save As.
  4. Choose a location and select CSV (Comma delimited) (*.csv) in the Save as type dropdown.
  5. Click Save and verify the file by opening it in a text editor.

Method 2: Google Sheets

  1. Open Google Sheets.
  2. Enter your data as in Excel.
  3. Click File → Download → Comma-separated values (.csv).
  4. The file will be saved to your computer.

Method 3: LibreOffice Calc

  1. Open LibreOffice Calc (or OpenOffice Calc).
  2. Enter your data.
  3. Click File → Save As.
  4. In the File Type dropdown, select Text CSV (.csv).
  5. Click Save and confirm delimiter settings (choose “Comma” as the separator).

Important Notes

  • Avoid extra commas: Ensure your data does not include unintended commas. Wrap text in "quotes" if needed.
  • Encoding Issues: Save as UTF-8 encoding if your data includes special characters.
  • Line Breaks: Avoid pressing Enter inside cells, as it may cause formatting issues.
